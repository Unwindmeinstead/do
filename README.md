# Do.

A minimal, beautiful card-based task manager.

## Features

- Casino-style stacked cards
- Smart task categorization
- Drag and drop functionality
- Beautiful animations
- PWA support

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Building for Production

```sh
npm run build
```
